Implementação de funções simples de desenho.

-  Um ponto;
-  Uma reta;
-  Um triângulo.

As funções de plot de reta e desenho de um triângulo usaram como base o algoritmo de bresenham.

Instruções
=============
#### Requisitos
* python >= 3.5

#### Instalando e rodando o projeto
`$ ./build; . venv/bin/activate; make run`
